5-About_Simple_Picker

走路草的博客
2018-01-15

Today we are going to learn about the design concept of the Simple Picker product.

Controllers - Enum Types

PickerGridItemType

An enum to define the media type that can be rendered in the simple picker by different views. (photo, video, gif and live camera)

EnabledMediaTypeUtil

Define a map from SupportedMediaType (commonly used) and the PickerGridItemType(used only in Simple Picker).

Controllers - Others

PickerSelectionController

Control the selection behavior of the picker.

PickerSelectionUpdateUtil

Remove the deleted items after restore from background.

SimplePickerDataLoadUtil

Used for reading the data from database. Note that we will create a placeholder at the first cell for displaying the live camera later.

SimplePickerGridViewUtil

Used to get the total number of grid items for the first page so that we know how many data to get when we ask database.

SimplePickerHighlightsController

Used for show the highlight of the selected images.

SimplePickerGridViewCursorAdapter

Expose the data from the cursor to the recycler view (Bind the media with the specific view).

SimplePickerRecyclerViewAdapter

Using SimplePickerGridViewCursorAdapter to perform adapter functions

SimplePickerViewControllerAdapter

Using SimplePickerRecyclerViewAdapter to perform functions.

SimplePickerViewController

Using SimplePickerViewControllerAdapter to perform functions. It is directly used in fragment to perfom functions.

SimplePickerItemHolder

View Holder used in the recycler view.

Views

PickerGridItemDraweeView

This is the base view for each media grid.

HasPickerHighlightLayer

This is an interface for each grid view to implement to display the faded out layer on the non-highlighted items.

SelectableView

This is the interface for each grid view type to implement.

PickerPlayableView

This is the interface for playable media grid view, e.g., video grid.

PickerGridGifItemDraweeView

This is the view for Gif media grid.

PickerGridLiveCameraItemDraweeView

This is the view for Live Camera media grid.

PickerGridPhotoItemDraweeView

This is the view for photo media grid.

PickerGridVideoItemDraweeView

This is the view for vide media grid.

GridItemViewOnTouchListener

Responsible for handling the long-press functionality.

PickerGridPhotoItemDraweeView

This is the view for photo media grid.

PickerGridVideoItemDraweeView

This is the view for vide media grid.

GridItemViewOnTouchListener

Responsible for handling the long-press functionality.

PickerLongPressProgressBar

The progress bar to indicate the progress of user’s long_perss.
